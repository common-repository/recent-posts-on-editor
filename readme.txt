Recent Posts on Editor - Displaying recent contents (post, page or custom post type) on WordPress editor.

== Description ==

Recent Posts on Editor is WordPress plugin to display recent contents (post, page or custom post type) on WordPress editor metabox. With this plugin you can review what you have just posted, and also you can simply insert those links into current post.


= Features =
* Easy to use
* Displays recent posts and pages
* Link to insert, edit or view
* Ajax search
* Filter by post status (publish, draft, etc)

= Other notes =
* Currently only support post and page post type.
* Insert link only works when you are on Visual tab.

== Installation ==

There are many ways to install this plugin, e.g:

1. Upload compressed (zip) plugin using WordPress plugin uploader.
2. Directly install from WordPress.org directory
3. Upload manually uncompressed plugin file using FTP.
